Topics

  1. Containers

    Consistent containers are important to help improve alignment, readability, and organization of content.

  2. Media Queries

    Media queries allow us to change the styles that are applied based on different factors such as screen size, dark/light mode, print, and more.

  3. Stylesheet

    A stylesheet is a good way to test out and create a reference for site wide styles.

  4. Responsive Type

    Lets make our typography responsive to make sure it creates good hierarchy and content organization on our web pages.

  5. Fluid Type

    Let's examine an alternative method for creating responsive type styles that take advantage of modern CSS techniques.
